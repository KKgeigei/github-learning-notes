# Day 01

Today I created my first GitHub account and my first repository.

## What I did today

- completed my profile
- created a public repository
- edited the README
- made my first commits
- viewed commit history
- learned how GitHub shows file changes

## What I learned

- A repository is like a project folder.
- README is the front page of a repository.
- A commit is a saved record of changes.
- GitHub keeps the history of each change.
- I can look at old and new versions through commit history.

## My takeaway

GitHub feels less scary now that I understand the basic flow:
create a repository, edit a file, commit changes, and view history.
